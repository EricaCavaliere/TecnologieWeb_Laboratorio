# Tecnologie Web (Laboratorio)
Sito web creato durante il corso di Tecnologie Web della prof.ssa Gaggi nell'anno 2022-23, UniPD, Informatica.

L'argomento alla base del sito è la presentazione della squadra di pallavolo maschile italiana, in onore per la loro vittoria al campionato mondiale di pallavolo nel 2022, tutte le immagini presenti sono state fornite dalla docente per solo scopo didattico.

Le pagine principali sono:
- HOME: riporta le informazioni più importanti sulla squadra durante il campionato (cioè l'argomento principale) e le ultime notizie di pallavolo (datate al 2022)
- L'allenatore: viene data una biografia dell'allenatore
- La squadra: vengono riportati i dati principali dei giocatori con eventuali note o riconoscimenti
- Il fairplay: viene spiegato che cos'è il fairplay nella pallavolo e vengono indicati un paio di fatti o eventi interessanti riguardo a questa regola

Il sito è stato creato con lo scopo di:
- imparare i linguaggi HTML, CSS, php e javascript
- imparare a creare un sito web accessibile e fruibile
- applicare la teoria imparata durante il corso

## Attenzione

Questo non è il sito principale della squadra di pallavolo, reperibile invece al link [federvolley.it](https://www.federvolley.it/). Questa repo è stata creata per un uso didattico e la utilzzerò per eventuali prove personali con il solo scopo di imparare il codice e la teoria che c'è dietro lo sviluppo di un sito web.
